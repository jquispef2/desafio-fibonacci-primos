01 — Contexto

El Problema Real

Muchas personas ahorran dinero de forma constante, pero eso puede sentirse monótono. ¿Y si tu ahorro siguiera el patrón de la naturaleza? La serie de Fibonacci aparece en las flores, las galaxias y el crecimiento de poblaciones. Aplicada al ahorro, permite aumentar gradualmente el dinero guardado cada mes de forma natural y progresiva.

Fibonacci en la naturaleza

Los pétalos de flores, las espirales de caracoles y las ramas de árboles siguen esta secuencia matemática.

Primos en seguridad

Los números primos son la base de la criptografía moderna. Detectarlos en tu ahorro es un bonus especial.

Ahorro progresivo

Cada mes ahorras la suma de los dos meses anteriores, haciendo el esfuerzo inicial pequeño y manejable.

02 — Algoritmo

¿Cómo funciona?

Serie de Fibonacci

La serie comienza con 1 y 1. Cada número siguiente es la suma de los dos anteriores:

1123581321…

Número Primo

Un número es primo si solo se divide exactamente entre 1 y sí mismo. Por ejemplo: 2, 3, 5, 13 son primos. El 4 no lo es porque también se divide entre 2.

script.js — Lógica principal
function fibonacci(n) {
  let a = 1, b = 1, c;
  let serie = [a, b];
  for (let i = 2; i < n; i++) {
    c = a + b;
    serie.push(c);
    a = b; b = c;
  }
  return serie;
}

function esPrimo(num) {
  if (num < 2) return false;
  for (let i = 2; i <= num; i++) {
    if (num % i === 0 && i !== num)
      return false;
  }
  return true;
}

04 — Conclusión

¿Cómo ayuda la matemática?

La serie de Fibonacci nos muestra que el crecimiento natural es gradual y sostenible. Al aplicarla al ahorro, comenzamos con cantidades pequeñas que se incrementan de forma orgánica, haciendo más fácil mantener el hábito. Los números primos, además, nos recuerdan que ciertos valores tienen propiedades matemáticas únicas, lo que los hace especiales tanto en seguridad como en patrones numéricos. Este proyecto demuestra que las matemáticas no son abstractas: están en nuestra economía personal, en la naturaleza y en la tecnología que usamos cada día.
